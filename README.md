# Physics of Complex Systems - Laboratory Projects

This repository contains a collection of computational models and simulations developed for the Physics of Complex Systems course at the AGH University of Science and Technology. The project focuses on the implementation of numerical methods to study emergent behavior, non-linear dynamics, and self-organization in various physical and social systems.

## Computational Approach

The projects implement a hybrid software architecture designed to optimize both computational throughput and data interpretability:

* **Kotlin (JVM):** The core simulation engines are implemented in Kotlin. This choice provides the necessary performance for high-frequency updates in agent-based models and large-scale lattice simulations while maintaining strict type safety and modern concurrency patterns (Coroutines).
* **Python:** Data post-processing, statistical validation, and scientific visualization are handled via Python. This integration allows for the use of advanced libraries such as NumPy and Matplotlib to analyze phase transitions and topological properties of the simulated systems.

## Scope of Research

The implementations cover a broad spectrum of complexity science topics, including:
* Discrete dynamical systems and cellular automata.
* Deterministic chaos and strange attractors.
* Topology and dynamics of complex networks.
* Stochastic processes in multi-agent systems.
